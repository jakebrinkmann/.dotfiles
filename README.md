# dotfiles
sensible hacker defaults for CentOS

## Installation 

Download/Install this project:
```bash
git clone git@github.com:jakebrinkmann/dotfiles.git ~/.dots
cd ~/.dots
bash setup.sh
```

### VIM

```bash
git clone https://github.com/VundleVim/Vundle.vim.git ~/.vim/bundle/Vundle.vim
vim # :BundleInstall
```
